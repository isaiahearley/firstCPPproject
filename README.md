# firstCPPproject
C++ Story Selector which allows me to familiarize myself with loops, variables, conditionals and flow control. 

> To summarize this program utilizes various RNG elements to try and fight the only enemy in the mini-game known as Bear.
You have the choice between 2 characters; Florence and Isabella who each have similar but also different abilities to choose 
from. I have refrained from using functions, classes, and so on to familiarize myself with loops, variables, conditionals, 
and flow control.
